In order to support forwards and backwards compatibility on all themes, Mable uses namespacing in order to ensure that no CSS classes will collide.

This only applies to the pattern ?*-?*, meaning that a namespace must only use classes with a - after its name, such as element-white. Mable uses other class selectors that are single words.

Here is the list of namespaces Mable currently uses:

  • section
  • element
  • container
  • col
  • cols
  • text
