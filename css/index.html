How to Install

1.1

GNU/Linux

I highly reccomend installing through your package manager (as root):

apt install gpg
pacman -S gpg

Please note you will only have to do one of those.

You can allways download the compiled code directly from gnupg.org as well.

I also reccomend installing "seahorse-nautilus" if you use the Nautilus file browser (default for Ubuntu and Gnome) since it allows you to encrypt and sign stuff from a GUI.

1.2

MacOS

Open the terminal application, paste in the following and press enter; it will ask for your password:

sudo brew install gpg

1.3

Windows

Download and install GPG4Win. The checked software in the installer should be what you need.

2.0

Navigating the Comand Line

This guide assumes you know how to reasonably use the following commands: cd, ls, mv, & rm. If you don't you should consult the manual for them by using the command man [command], or by looking up a guide. Those four specifically work across all platforms.

2.1

GPG Syntax

GPG has a very simple layout of the following:

gpg [flags & options] <object filename>

I will allways use that terminoligy exactly.
